**Simplicity is measured by ease of reassembly.** My working description of achieving simplicity in practice: "pull the system apart, and build it back together again, many times over."

If components can be recombined in new ways without modification, they're simple. If reassembly requires rewriting, they're tangled. The test: can you build something new from existing pieces?
