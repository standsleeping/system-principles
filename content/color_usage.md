---
id: COLOR_USAGE
title: "Color Usage."
essence: "A restrained palette makes intentional color usage more impactful; reserve color for meaning."
---

Reserve color for meaning. When most of the interface is neutral, the appearance of color immediately signals something worth noticing: a status change, an interactive affordance, an error.

1. Use color to encode meaning (status, interaction, severity), not to decorate
2. Keep the default palette neutral so that intentional color is visually distinct
3. Avoid using background color as the sole differentiator between categories

A restrained palette makes intentional color usage more impactful.
