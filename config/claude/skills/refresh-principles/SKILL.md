# Refresh Principles

Recompile design principles from system-principles into Claude Code config files (fragments and rules).

## Steps

1. Run the export command from the system-principles directory:
   ```
   uv run --directory <system-principles> principles export
   ```

2. Show the user what was generated by reading the output files and reporting their line counts.

3. If any errors occurred, report them clearly.

Replace `<system-principles>` with the system-principles repository path from your context.
